Details of the Python editor toolbar:

- |new_python| button is used to create a new tab in the editor.
- |save_python| button is used to save the current tab in the editor.
- |clear_python| button is used to clear the output pan.
- Run button |run_button| is used to execute a program.
- Run-load button |run_load_button| is used to execute the program and load the generated files, if any.
- |kill_python| button kills a running Python instance.
  This button will be enabled only if a Python instance is running in that tab, else will be disabled.
- Clicking |bg_python| button, background and font color for CINEMAS-Python editor can be set, separately for editor and output pan.
